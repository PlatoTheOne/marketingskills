岗位角色与工作流模块

结论先说：一个 skill 不是一个完整工作流。它更像单点能力；真正可执行的工作流通常由 product-marketing-context + 多个 skills 组合而成。
你后续只要说工作流 ID（例如 WF-SEO-05），我就可以按该链路自动展开执行步骤、产出模板和优化建议。

角色 1：SEO 与内容增长负责人

WF-SEO-01

站内 SEO 修复冲刺

用于技术 SEO、结构化数据、站点结构的一次性修复。

product-marketing-context -> seo-audit -> site-architecture -> schema-markup -> analytics-tracking

调用：WF-SEO-01
WF-SEO-02

对外内容建设（权威内容矩阵）

围绕核心主题建立外部可传播内容，提升品牌与搜索可见性。

product-marketing-context -> content-strategy -> copywriting -> social-content -> ai-seo

调用：WF-SEO-02
WF-SEO-03

外链建设与外联（Backlink Ops）

通过对比页、外联邮件、合作资产，系统化获取外链与提及。

product-marketing-context -> competitor-alternatives -> cold-email -> sales-enablement -> revops

调用：WF-SEO-03
WF-SEO-05

PSU/pSEO 内容工厂

规模化页面生产、模板数据联动、索引与转化闭环。

product-marketing-context -> programmatic-seo -> schema-markup -> ai-seo -> analytics-tracking

调用：WF-SEO-05

安装方式

README 中的 6 种安装方式继续沿用。

OpenClaw 自然语言安装（推荐）
请帮我安装 coreyhaines31/marketingskills 的全部 skills。安装完成后列出已安装 skills，并验证可用性。
安装全部 skills
npx skills add coreyhaines31/marketingskills
安装指定 skills
npx skills add coreyhaines31/marketingskills --skill page-cro copywriting
Claude 插件安装
/plugin marketplace add coreyhaines31/marketingskills && /plugin install marketing-skills
SkillKit 安装
npx skillkit install coreyhaines31/marketingskills

01

CLI（推荐）

通过 npx skills 安装全部或指定 skill。

npx skills add coreyhaines31/marketingskills
npx skills add coreyhaines31/marketingskills --skill page-cro copywriting
npx skills add coreyhaines31/marketingskills --list

02

Claude Code 插件

先添加 marketplace，再安装 marketing-skills。

/plugin marketplace add coreyhaines31/marketingskills
/plugin install marketing-skills

03

Clone + Copy

克隆仓库后复制 skills 到 .agents/skills/。

git clone https://github.com/coreyhaines31/marketingskills.git
cp -r marketingskills/skills/* .agents/skills/

04

Git Submodule

适合团队维护与统一更新。

git submodule add https://github.com/coreyhaines31/marketingskills.git .agents/marketingskills

05

Fork 定制

Fork 后按业务场景改造再接回项目。

1. Fork this repository
2. Customize skills for your needs
3. Clone your fork into projects

06

SkillKit（多代理）

适配 Claude Code、Cursor、Copilot 等多代理环境。

npx skillkit install coreyhaines31/marketingskills
npx skillkit install coreyhaines31/marketingskills --skill page-cro copywriting
npx skillkit install coreyhaines31/marketingskills --list
